    HTML CSS

    Klik untuk kembali ke bagian atas blog

    Dengan CSS, Anda dapat mengontrol warna, font, ukuran teks, jarak antar elemen, bagaimana elemen diposisikan dan ditata, gambar latar belakang atau warna latar belakang apa yang akan digunakan, tampilan berbeda untuk perangkat dan ukuran layar berbeda, dan banyak lagi!

    1. Inline CSS

    Inline CSS digunakan dengan menambahkan atribut style langsung ke elemen HTML. Contoh:

    <p style="color: red; font-size: 20px; text-align: center;">Teks ini menggunakan Inline CSS</p>

    Hasilnya adalah teks dengan warna merah, ukuran font 20px, dan posisi teks di tengah seperti di bawah:.

    Teks ini menggunakan Inline CSS

    2. Internal CSS

    Internal CSS digunakan dengan menambahkan tag <style> di dalam tag <head> pada dokumen HTML. Contoh:

    <head>
    <style>
    p {
    color: blue;
    font-size: 20px;
    text-align: center;
    }
    </style>
    </head>
    <body>
    <p>Teks ini menggunakan Internal CSS</p>
    </body>

    Hasilnya adalah teks dengan warna biru, ukuran font 20px, dan posisi teks di tengah seperti di bawah:

    Teks ini menggunakan Internal CSS

    3. External CSS

    External CSS digunakan dengan membuat file CSS terpisah dan menghubungkannya ke dokumen HTML menggunakan tag <link> di dalam tag <head>. Contoh:

    <head>
    <link rel="stylesheet" href="style.css">
    </head>

    File CSS (style.css):

    code di dalam file CSS:
    p {
    color: green;
    font-size: 20px;
    text-align: center;
    }

    Hasilnya adalah teks dengan warna hijau, ukuran font 20px, dan posisi teks di tengah seperti di bawah:

    Teks ini menggunakan External CSS

    4. CSS Border

    CSS border digunakan untuk menambahkan garis di sekitar elemen. Contoh:

    <p style="border: 2px solid black;">Teks ini di dalam border</p>

    Hasilnya adalah teks di dalam border hitam seperti di bawah:

    Teks ini di dalam border


    CSS border juga bisa di gunakan untuk memberi warna pada border gambar

    contoh code:

    <img src="namafile gambar" alt="keterangan gambar jika tidak muncul" style="border: 2px solid red;">

    maka hasilnya akan seperti di bawah:


    5. CSS Padding

    CSS padding di gunakan untuk menambahkan luas isi suatu elemen. Contoh:

    <p>teks tanpa padding<p>
    <p style="padding: 20px; background-color: blue;">Teks ini memiliki padding 20px</p>
    <p>teks tanpa padding<p>

    Hasilnya adalah teks dengan padding 20px dan background biru seperti di bawah:

    teks tanpa padding

    Teks ini memiliki padding 20px

    1. HTML Links - Syntax

    HTML links digunakan untuk menambahkan tautan ke halaman web lain atau ke bagian lain dari halaman yang sama.

    Contoh code:

    <a href="https://www.instagram.com/ex1mks">Instagram Eki Mukhlis</a>
    <a href="#top">Kembali ke atas</a>

    Hasilnya adalah tautan ke Instagram Eki Mukhlis seperti di bawah:

    Instagram Eki Mukhlis
    Kembali ke atas

    2. HTML Links - The target Attribute

    HTML links juga dapat menggunakan atribut target untuk menentukan di mana halaman yang ditautkan akan dibuka.

    Contoh code:

    <a href="https://www.instagram.com/ex1mks" target="_blank">Instagram Eki Mukhlis</a>

    Hasilnya adalah tautan ke Instagram Eki Mukhlis yang akan dibuka di tab baru:

    Instagram Eki Mukhlis

    3. HTML Links - Use an Image as a Link

    HTML links juga dapat menggunakan gambar sebagai tautan.

    Contoh code:

    <a href="https://www.instagram.com/ex1mks">

    </a>

    Hasilnya adalah gambar yang dapat diklik untuk membuka tautan ke Instagram Eki Mukhlis:


    4. Link to an Email Address

    HTML links juga dapat digunakan untuk membuat tautan ke alamat email.

    Contoh code:

    <a href="mailto:namaemai@gmail.com">Kirim email</a>

    Hasilnya adalah tautan yang dapat diklik untuk mengirim email:

    Kirim Email

    5. Button as a Link

    HTML links juga dapat menggunakan tombol sebagai tautan.

    HTML Tables

    Klik untuk kembali ke bagian atas blog

    1. Table

    HTML tables digunakan untuk menampilkan data dalam bentuk tabel.

    Contoh code:

    <table border="1" cellpadding="5" cellspacing="0">
    <tr>
    <th>Nama</th>
    <th>Umur</th>
    </tr>
    <tr>
    <td>Eki Mukhlis</td>
    <td>20</td>
    </tr>
    </table>

    Hasilnya adalah tabel dengan 2 kolom dan 1 baris data:

    Nama Umur
    Eki Mukhlis 20

    2. Table Colspan dan Rowspan

    HTML colspan digunakan untuk menggabungkan beberapa kolom, sedangkan rowspan digunakan untuk menggabungkan beberapa baris dalam tabel.

    Contoh Colspan:

    Code:

    <table border="1" cellpadding="5" cellspacing="0">
    <tr>
    <th colspan="2">Header Gabungan</th>
    </tr>
    <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    </tr>
    </table>

    Hasilnya:

    Header Gabungan
    Data 1 Data 2

    Contoh Rowspan:

    Code:

    <table border="1" cellpadding="5" cellspacing="0">
    <tr>
    <th>Header 1</th>
    <th>Header 2</th>
    </tr>
    <tr>
    <td rowspan="2">Data Gabungan</td>
    <td>Data 2</td>
    </tr>
    <tr>
    <td>Data 3</td>
    </tr>
    </table>

    Hasilnya:

    Header 1 Header 2
    Data Gabungan Data 2
    Data 3

    HTML List

    Klik untuk kembali ke bagian atas blog

    1. Unordered List

    HTML unordered list digunakan untuk menampilkan daftar tanpa urutan tertentu.

    Contoh code:

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    </ul>

    Hasilnya adalah daftar tanpa urutan tertentu:

    • Item 2
    • Item 3

    2. Ordered List

    HTML ordered list digunakan untuk menampilkan daftar dengan urutan tertentu.

    Contoh code:

    <ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    </ol>

    Hasilnya adalah daftar dengan urutan tertentu:

    1. Item 1
    2. Item 2
    3. Item 3
